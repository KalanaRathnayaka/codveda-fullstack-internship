# Level 2 - Task 1: Frontend with a JavaScript Framework

## Task Description
This task was completed by recreating the frontend using React.js.  
The frontend connects with the Node.js Express REST API and performs CRUD operations for products.

## Technologies Used
- React.js
- Vite
- Axios
- CSS
- Node.js REST API
- Express.js Backend

## Features Implemented
- React project setup using Vite
- Reusable components
- Product form component
- Product card component
- API service file using Axios
- Fetch products from backend API
- Add new product
- Edit existing product
- Delete product
- Loading state handling
- Responsive professional dashboard UI

## Components Created

### ProductForm.jsx
Used to add and update product details.

### ProductCard.jsx
Used to display each product with edit and delete buttons.

### productService.js
Used to manage API calls between React frontend and Express backend.

## API Connection
Frontend API URL:

http://localhost:5000/api/products

## Pages / UI
The frontend contains:
- Header section
- Total products card
- Feature cards
- Add product form
- Product list section
- Edit and delete actions
